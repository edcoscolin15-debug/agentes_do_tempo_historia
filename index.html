⏳ Agentes do Tempo: A Missão

Agentes, atenção! A continuidade do tempo está quebrado e a nossa linha histórica corre perigo. Vocês foram convocados por serem os melhores especialistas em História e a sua missão é clara: precisam restaurar a ordem, alinhando os fatos exatamente como eles ocorreram. O destino do passado está nas mãos de vocês!

📜 Regras da Missão

  • Organização Cronológica: Na zona de desafio, os cards dos eventos devem ser ordenados do mais antigo (topo) para o mais recente (base).
  • O Limite do Agente: Cada time dispõe de 60 segundos e 2 corações (vidas) para validar a resposta.
  • Fim de Jogo: Se o tempo esgotar ou as vidas acabarem, o sistema é travado. Os cards corretos permanecerão travados em suas posições exatas.
  • Recompensa Parcial: O professor irá pontuar manualmente o time de acordo com os cards posicionados corretamente.

📖 Guia Prático: Agentes do Tempo Pro

Este roteiro foi desenhado para que você domine todas as funcionalidades, garantindo que suas aulas sejam dinâmicas e que seus dados estejam sempre seguros.

1. Painel Inicial (Menu Principal)

Ao abrir o arquivo no seu navegador (Chrome ou Edge), você encontrará quatro botões principais:

  • 🎮 Jogar: Acesso à lista de temas disponíveis para iniciar uma partida.
  • 🛠️ Criar ou Alterar Desafios: Acesso ao "Motor de Criação" (Editor).
  • 📥 Exportar Dados: Painel para gerar seus arquivos de backup.
  • 📤 Importar: Carrega arquivos .json que você criou anteriormente.

2. O Ambiente do Jogo (Em Ação)

Ao iniciar uma partida no seu tablet/celular Android, você verá o seguinte ambiente:

  • Placar Dinâmico: Use os botões "+ Pontos" para registrar o progresso dos times A, B, C ou D em tempo real.
  • Cronômetro (60s): O tempo começa a contar automaticamente. Se chegar a zero, o jogo pausa, o sistema bloqueia as ações e você pode liberar o Gabarito.
  • Vidas (2 Corações): A cada erro na ordenação, uma vida é perdida. Isso adiciona uma camada de estratégia: os times precisam pensar bem antes de validar.
  • Timeline (Timeline Zone): Onde os desafios aparecem embaralhados.
  • Usabilidade Android: Basta tocar e arrastar o card para cima ou para baixo para reordená-los. O sistema foi otimizado para que o toque não falhe.
  • Verificar Resposta: O botão mágico. Se a ordem estiver correta, os cards ficam verdes e o ano é revelado. Se errada, ficam vermelhos e uma vida é subtraída.
  • Mostrar Ordem Correta: Só aparece após as vidas acabarem ou o tempo esgotar. É a ferramenta perfeita para a explicação final da matéria.

3. Editor de Conteúdos e Desafios

Aqui é onde você transforma o jogo no que você precisa para sua aula:

  • Criar Novo Tema: Digite o nome (ex: "Idade Média") e clique em "Criar Tema". O sistema salvará isso automaticamente.
  • Filtragem Inteligente: Use o menu suspenso para focar apenas no conteúdo da semana. Isso evita que você precise navegar por dezenas de perguntas antigas.
  • 🗑️ Excluir Tema: Caso precise limpar o ambiente, selecione o tema no filtro e clique no botão "🗑️ Excluir Tema". Atenção: isso apagará o tema e todos os desafios atrelados a ele.
  • Adicionar/Editar Desafios: Escolha o tema, escreva o enunciado e preencha os 4 campos de eventos em ordem cronológica (sempre da data mais antiga para a mais nova). Clique em "Salvar Alterações". O sistema salvará tudo na memória do navegador do tablet.

4. Exportação e Importação (.json)

Esta é a função que garante que você nunca perca o seu trabalho:

  • Exportar: Escolha se quer baixar "Tudo" ou apenas um "Tema Específico". O tablet fará o download de um ficheiro .json. Esse ficheiro é o seu "cartucho de jogo" — pode guardá-lo no Google Drive ou enviar por e-mail.
  • Importar: Se você for para outra escola ou trocar de tablet, basta clicar em "Importar", selecionar o ficheiro .json e o jogo será montado exatamente como você deixou.

📱 Usabilidade no Android (Tablet ou Celular)

O Agentes do Tempo foi otimizado para o ecossistema Android:

  • Offline: Uma vez que você tenha o ficheiro .html no dispositivo, não precisa de internet. A gincana funciona no meio do pátio ou na sala de aula sem sinal.
  • Salvamento Automático (LocalStorage): Se a bateria acabar ou o navegador fechar, não se preocupe. O Android guarda as alterações na memória interna. Ao reabrir o ficheiro, tudo estará lá.
  • Toque Preciso: O sistema de arrastar e soltar (Drag and Drop) foi ajustado para telas táteis. Ao tocar no card, ele entra em modo de edição/movimentação; ao soltar, ele fixa na nova posição.
  • Downloads Nativos: Ao exportar, o Android Chrome tratará o ficheiro como um download legítimo, enviando-o para a pasta "Downloads" do seu tablet, pronta para ser compartilhada.

💡 Dica de mestre: Mantenha sempre uma cópia dos seus arquivos .json no Google Drive. Assim, se você trocar de tablet, basta fazer o download e importar novamente em menos de 10 segundos!

⚠️ ATENÇÃO

O arquivo .html é só o motor do jogo (a estrutura, o visual, as regras). As perguntas que você cria ficam na memória do navegador. E os arquivos .json que você exporta são o seu verdadeiro banco de dados (os cartuchos de jogo) que você guarda para não perder nada, então após criar um tema e os desafios, exporte para sua segurança e importe quando conveniente.

Se você quiser que uma pergunta nova faça parte do arquivo .html para sempre (sem precisar importar), aí sim você precisaria abrir o .html no Bloco de Notas e digitar o código lá dentro. Mas, com o sistema de exportar/importar .json que criamos, você não vai precisar mexer em código.

Se preferir agilizar a criação de conteúdos (temas) e os desafios, pode solicitar para a IA montar em arquivo JSON (DICA: envie um modelo exportado para a IA analisar a estrutura do código JSON). Após ela montar, basta fazer o download (ou copiar o código e colar em um Bloco de Notas ou NotePad++ e salvar com o nome do “conteúdo.json” seguindo o formato, exemplo IDADE_MEDIA.json).

Lembre de solicitar os critérios:
- Dificuldades misturadas (fácil, média e difícil).
- Todas valem 1 ponto. (ou pontuar de acordo com o nível de dificuldade, porém toda dificuldade valer 1 torna mais justo)
- Ordem totalmente aleatória.

🤖 Prompt para a IA:

Copie e cole o texto abaixo no ChatGPT ou Gemini para gerar novas perguntas automaticamente:

Aja como um especialista em desenvolvimento de jogos educacionais. Preciso que crie um novo arquivo JSON para o jogo 'Agentes do Tempo', seguindo rigorosamente a estrutura do arquivo. Requisitos: - Tema da atividade: [INSIRA O TEMA AQUI, EX: Independência do Brasil] - Quantidade de perguntas: [INSIRA A QUANTIDADE, EX: 24] - Estrutura: O JSON deve conter os campos versao, listaConteudos e o array bancoFases. Cada item em bancoFases deve ter: conteudo, pergunta, nivel, pontos e um array de eventos. - Regra de Eventos: Cada evento dentro do array eventos deve conter obrigatoriamente: id (formato único, ex: 'id_1'), ano, titulo, desc (descrição detalhada) e ordem (índice numérico sequencial). - Nível de dificuldade: Equilibre entre Fácil, Médio e Difícil para os itens do bancoFases. Formato de Saída: - Forneça apenas o código JSON pronto para ser copiado. - Não escreva introduções, explicações ou comentários fora do código. - Garanta que o JSON esteja sintaticamente correto, com todas as vírgulas e chaves fechadas corretamente, pronto para ser importado no meu projeto.

Dicas para melhores resultados:

  • Customização: Quando for usar o prompt, substitua o que está entre colchetes pelos dados do seu próximo desafio histórico.
  • Se o tema for complexo: Se você precisar que a IA crie os eventos baseados em fatos reais de um período, adicione ao prompt: "Utilize fatos históricos reais e precisos para compor os eventos e as descrições de cada fase do tema [INSIRA O TEMA AQUI]."
  • Validação: Se a IA gerar um código muito longo e ele parecer cortado, basta digitar "Continue de onde parou" ou "Forneça o JSON completo sem truncar".
